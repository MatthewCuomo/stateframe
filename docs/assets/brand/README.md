# stateframe Brand Assets

These assets use the "ledger aperture" concept: an optic/data-science control
surface for adaptive exploratory data analysis with a branching, git-style
analysis ledger inside it.

- `stateframe-mark.svg` is the compact square mark for favicons, package icons,
  app launchers, and small README placements.
- `stateframe-logo.svg` is the horizontal lockup for light backgrounds.
- `stateframe-social-preview.svg` is the source file for the GitHub repository
  social preview.
- `stateframe-social-preview.png` is the upload-ready GitHub social preview export.

The GitHub social preview should be uploaded from the repository settings under
Social preview. GitHub recommends PNG, JPG, or GIF files under 1 MB.
